Das Datei-Menü

  • Neu

    Dieser Menüpunkt öffnet ein neues Anwendungsfenster. Es befinden sich zu Beginn keine Punkte im neuen Fenster. Das alte Fenster bleibt dabei geöffnet.

  • Laden und ersetzen

    Mit diesem Menüpunkt lassen sich neue Punkte aus einer Datei einlesen. Dabei werden alle Punkte, die aktuell im Editor angezeigt werden, durch die neuen Punkte ersetzt. Nach dem Klick auf den Menüpunkt öffnet sich ein Dateiauswahldialog, in dem die gewünschte Datei auszuwählen ist. Es kann jede beliebige Datei ausgewählt werden. Allerdings muss diese Datei von der Anwendung lesbar sein - es sollte sich also um eine Datei handeln, die vorher mit dieser Anwendung angelegt wurde.

  • Laden und hinzufügen

    Dieser Menüpunkt funktioniert wie "Laden und ersetzen", allerdings werden hier nicht alle Punkte im Editorfenster ersetzt. Die neuen Punkte werden zu den vorhandenen hinzugefügt.

  • Speichern

    Hiermit lassen sich die aktuell im Editorfenster angezeigten Punkte auf dem Dateisystem abspeichern. Sollten die angezeigten Punkte aus einer Datei mit "Laden und ersetzen" geladen worden sein, wird diese Datei sofort überschrieben. Ansonsten öffnet sich ein Dateiauswahl-Dialog, in welchem der gewünschte Speicherort angegeben werden kann. Wird eine bereits vorhandene Datei ausgewählt, so wird diese überschrieben!

  • Speichern unter

    Im Gegensatz zum Menüpunkt "Speichern" wird hier immer ein neuer Speicherort ausgewählt, an dem die Punktdatei angelegt werden soll.

  • Schließen

    Mit diesem Menüpunkt wird das Anwendungsfenster geschlossen. Handelt es sich um das letzte noch geöffnete Fenster, so wird die Anwendung beendet. Alle nicht gespeicherten Änderungen im Fenster gehen dabei verloren!
