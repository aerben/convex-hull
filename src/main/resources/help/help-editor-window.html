Das Editorfenster

Dies ist der Hauptarbeitsbereich der Anwendung. Es lassen sich bestehende Punkte löschen und verschieben sowie neue Punkte hinzufügen. Dabei wanderen die Hüllen um die Punktmenge stets mit. Die Punkte, die auf der konvexen Hülle liegen, sind farbig markiert. Die Winkelhüllen sind rot umrandet.

Das Editorfenster lässt sich in drei verschiedene Modi schalten:

  • Zeichnen

    In diesem Modus bewirkt jeder Klick in das Editorfenster, dass ein neuer Punkt an der gewählten Stelle erscheint.

  • Bewegen

    Dieser Modus aktiviert Drag-and-Drop auf dem Editorfenster, sodass bestehende Punkte über das Fenster gezogen werden können. Die Hüllen wandern dabei mit, jedoch wird die Bewegung erst nach Loslassen der Maustaste gespeichert. Ein Klick auf Rückgängig im Bearbeiten-Menü setzt den Punkt dann auf den Ort zurück, wo die Maus gedrückt wurde

  • Löschen

    In diesem Modus wird jeder Punkt, der auf dem Editorfenster angeklickt wird, gelöscht.
